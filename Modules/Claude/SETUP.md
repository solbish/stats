# Quick Setup Guide

## Option 1: Web Cookies (Recommended)

```
1. Install "Cookies Extractor" Chrome extension
2. Go to claude.ai and log in
3. Click extension > "Copy as JSON Cookie"
4. Stats > Settings > Claude > "Import from Clipboard"
```

## Option 2: Claude Code CLI

```
1. Install: npm install -g @anthropic-ai/claude-code
2. Login: claude login
3. Stats > Settings > Claude > "Import from Keychain"
```

## That's it!

The Claude module will appear in your menu bar showing your usage percentage.
Click it to see detailed stats including:
- 5-hour and 7-day usage
- Reset countdowns
- 24-hour history
- Peak hours status
